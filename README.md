# SteelPoint Portfolio
-**Designed and developed by Jayden Foster.**

Modern personal portfolio built with React and Vite.

## Overview

This project serves as the foundation for showcasing my development work.  
It is designed with a clean, minimal layout and structured for easy updates and future expansion.

## Tech Stack

- React
- Vite
- Modern CSS
- Component-based architecture

## Features

- Centered hero layout with brand-focused design
- Reusable project card components
- Separated content and layout for maintainability
- Responsive grid system
- Custom styling aligned with brand identity

## Architecture Decisions

- **Component-based structure** for scalability
- **Default exports** for single-component files
- **Static assets served from `/public`**
- **Content separated into a data file** for maintainability
